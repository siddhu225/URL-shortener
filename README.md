# url-shortener

This is a ur shortener system made in Vue.js, the following technologies were used:

* Vue.js 3 for Frontend
* Node.js for backend
* express js
* Bulma for css
* MongoDB for Database.
* Axios
* font-awesome 
* Vue-Router


## Project setup
```
npm install
```
